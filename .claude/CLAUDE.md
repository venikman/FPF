# FPF Project Rules

## File Modification Policy

**Allowed to modify:**
- `Readme.md`
- `.github/` (workflows, templates, CI/CD)

**Do NOT modify:**
- `FPF-Spec.md` — primary specification (maintained by author)
- `docs/` — specification parts (auto-generated from FPF-Spec.md)
- `prompts/` — prompt templates

The specification files are maintained by the primary author and should not be changed via PRs.
